# AI-Based Smart Document Verification System

## Overview
This project is designed to automate the document verification process using Artificial Intelligence techniques. It focuses on ensuring the integrity and authenticity of documents through various computational methods.

## Features
- Document Verification through Computer Vision (CV)
- Optical Character Recognition (OCR) for text extraction
- Natural Language Processing (NLP) for content validation
- Face Verification for identity confirmation

## Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/Asjath633/NEW_AI.git
   ```
2. Navigate to the project directory:
   ```bash
   cd NEW_AI
   ```
3. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```

## Usage
To run the application:
```bash
python app.py
```
